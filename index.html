Projeto: Busca de Filmes (APIs OMDb e TMDb)

Este projeto combina a busca simples OMDb (exercício de aula) com a busca avançada (filtros, detalhes e elenco) da nova API TMDb.

Etapa 1.A — Busca Simples (OMDb)

Digite o nome de um filme e clique em Buscar.

Etapa 1.B — Busca com Filtros (TMDb)

Use os filtros avançados da TMDb para buscar por Gênero ou Ano e veja os detalhes do filme.

Etapa 1.C — Buscar por Ator (TMDb)

Digite o nome completo de um ator para ver seus filmes mais populares.
